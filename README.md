## 简介
project-club是一种搭建前端项目架构方式。

[![js-standard-style](https://img.shields.io/badge/code%20style-standard-brightgreen.svg)](http://standardjs.com/)

## 技术栈

- webpack + babel
- koa + mongodb

## 安装依赖
```
npm install
```

## 开发
```
npm start
```

## 压缩
```
npm run build
```

## release
```
npm run release
```

## 查看更多定制
```
node app -h
```
